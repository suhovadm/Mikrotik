Сказания о том, как настроить fasttrack connection, т.е. функцию программного ускорения
обработки трафика за счёт оптимизации логики работы.

Всё это происходит на основном процессоре устройства, просто самым эффективным из возможных программных способов.

Fasttrack — это не магия «железа», а очень умное и эффективное программное решение для ускорения работы файрвола и снижения нагрузки на процессор.
